# mcp-1c-price

MCP-сервер для поиска программных продуктов 1С и формирования коммерческих предложений прямо из IDE.

Менеджер пишет свободным текстом — AI находит продукты в актуальном прайсе и возвращает готовую таблицу с кодами, ценами и итогом.

**Работает в:** Claude Code · OpenCode · Codex

## Быстрый старт

**1. Прописать сервер в IDE** — `uvx` скачает пакет автоматически. Пример для Claude Code (`.claude/settings.json`):

```json
{
  "mcpServers": {
    "1c-price": {
      "command": "uvx",
      "args": ["--from", "git+https://github.com/snn02/1c-products-price", "mcp-1c-price"]
    }
  }
}
```

**2. Загрузить прайс** — один раз после подключения:

```
обнови прайс
```

**3. Работать.**

Готовые конфиги для Claude Code, OpenCode и Codex — в папке [`configs/`](configs/).  
Подробнее — в [руководстве по настройке](docs/configuration.md).

## MCP-инструменты

| Инструмент | Что делает |
|-----------|-----------|
| `search_products(query)` | Поиск по названию (FTS5 + LIKE fallback) |
| `get_product(code)` | Получить продукт по артикулу |
| `build_quote(items)` | Сформировать КП: таблица + итог |
| `refresh_prices()` | Обновить базу из актуального прайса 1С |

Обновляется ежедневно. Обновить базу: `mcp-1c-price-update` или вызов `refresh_prices()` из IDE.
