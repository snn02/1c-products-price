# Установка и первый запуск

## Требования

- Python 3.12+
- [uv](https://docs.astral.sh/uv/) (рекомендуется) или pip

---

## Установка

### Вариант 1 — uvx из GitHub (рекомендуется, ничего не устанавливает глобально)

Прописать сервер в конфиг IDE — `uvx` скачает и запустит пакет автоматически при первом обращении. Никаких дополнительных команд не нужно.

Перейдите сразу к разделу [Настройка в IDE](configuration.md).

### Вариант 2 — глобальная установка через uv

```bash
uv tool install git+https://github.com/snn02/1c-products-price.git
```

### Вариант 3 — через pip

```bash
pip install git+https://github.com/snn02/1c-products-price.git
```

### Вариант 4 — локально из исходников

```bash
git clone https://github.com/snn02/1c-products-price.git
cd 1c-products-price
uv pip install -e .
```

---

## Первый запуск

### 1. Загрузить прайс-лист

Если устанавливали глобально (варианты 2–4):

```bash
mcp-1c-price-update
```

Если используете `uvx` (вариант 1) — загрузить прайс можно прямо из IDE:

```
обнови прайс
```

AI вызовет инструмент `refresh_prices()` автоматически.

Команда скачает актуальный прайс с сервера 1С (~580 КБ) и загрузит ~7 300 продуктов в локальную базу `~/.mcp-1c-price/prices.db`.

Вывод при успехе:

```
Прайс-лист обновлён.
Дата прайса: 28 апреля 2026
Загружено продуктов: 7329
Время обновления: 4.2 сек.
```

### 2. Прописать сервер в IDE

Смотрите [Настройка в IDE](configuration.md).

---

## Обновление прайса

Прайс-лист 1С обновляется ежедневно. Для обновления базы:

```bash
# Через CLI
mcp-1c-price-update

# Или попросите AI в IDE:
# "обнови прайс"
```

---

## Структура данных

База `~/.mcp-1c-price/prices.db` содержит продукты со следующими полями:

| Поле | Описание |
|------|----------|
| `code` | Артикул (13-значный код) |
| `name` | Полное наименование |
| `currency` | Валюта (руб., USD, BYN, KZT) |
| `price_retail` | Рекомендованная цена |
| `price_dealer` | Цена для дилера |
| `price_partner` | Цена для постоянного партнёра |
| `vat` | Ставка НДС |
| `section` | Раздел прайса |
| `date_start` | Дата начала продаж |
| `registry` | Номер в реестре российского ПО |
| `comment` | Комментарий |
