# MCP-инструменты

Сервер предоставляет четыре инструмента. AI в IDE вызывает их автоматически в зависимости от запроса менеджера.

---

## search_products

Поиск продуктов по свободному тексту. Использует полнотекстовый поиск (FTS5) с нечётким матчингом.

**Параметры:**

| Параметр | Тип | По умолчанию | Описание |
|---------|-----|-------------|----------|
| `query` | string | обязательный | Поисковый запрос (название, часть названия) |
| `limit` | int | 10 | Максимальное количество результатов |

**Пример вызова:**
```
query = "бухгалтерия 8 базовая"
limit = 5
```

**Пример ответа:**
```json
[
  {
    "code": "4601546057532",
    "name": "1С:Бухгалтерия 8. Базовая версия. Электронная поставка",
    "price_retail": 3600.0,
    "currency": "руб.",
    "vat": "Без НДС",
    "section": "Раздел 1. 1С:ПРЕДПРИЯТИЕ"
  },
  ...
]
```

## get_product

**Параметры:**

| Параметр | Тип | Описание |
|---------|-----|---------|
| `code` | string | Артикул продукта (13-значный код) |

**Пример вызова:**
```
code = "4601546057532"
```

**Пример ответа:**
```json
{
  "code": "4601546057532",
  "name": "1С:Бухгалтерия 8. Базовая версия. Электронная поставка",
  "currency": "руб.",

  "price_partner": 1620.0,

  "section": "Раздел 1. 1С:ПРЕДПРИЯТИЕ",
  "date_start": "01.01.2010",

  "comment": ""
}
```

## build_quote

Сформировать коммерческое предложение по списку позиций с количеством. Для каждой позиции выполняет поиск, берёт наилучшее совпадение и рассчитывает итог.

**Параметры:**

| Параметр | Тип | Описание |
|---------|-----|---------|
| `items` | array | Список позиций |
| `items[].query` | string | Название или описание продукта |
| `items[].qty` | int | Количество |

**Пример вызова:**
```json
{
  "items": [
    {"query": "ERP корп", "qty": 1},
    {"query": "сервер 1С мини", "qty": 1},
    {"query": "клиентская лицензия на 5 пользователей", "qty": 1}
  ]
}
```

**Пример ответа:**
```
| # | Код           | Наименование                                      | Кол-во | Цена      | Сумма      |
|---|---------------|---------------------------------------------------|--------|-----------|------------|
| 1 | 4601546064462 | 1С:ERP Управление предприятием 2. Корп            | 1      | 360 000 ₽ | 360 000 ₽  |
| 2 | 4601546100771 | 1С:Предприятие 8. Сервер МИНИ на 5 подключений   | 1      |  36 000 ₽ |  36 000 ₽  |
| 3 | 4601546057617 | 1С:Предприятие 8. Клиентская лицензия на 5 р.м.  | 1      |  21 600 ₽ |  21 600 ₽  |
|   |               |                                                   |        | ИТОГО:    | 417 600 ₽  |

⚠️ Требует уточнения:
- "ERP корп" — найдено 3 варианта, выбран наиболее популярный. Уточните редакцию если нужно.
```

## refresh_prices

Обновить базу данных из актуального прайс-листа 1С. Скачивает ZIP, парсит XLS, перезаписывает данные.

**Параметры:** нет

**Пример ответа:**
```
Прайс-лист обновлён.
Дата прайса: 28 апреля 2026
Загружено продуктов: 7 329
Время обновления: 4.2 сек.
```

Если прайс не изменился с последнего обновления:
```
Прайс актуален (обновлён сегодня в 03:08). Импорт пропущен.
```

## Сценарий использования

Менеджер пишет в IDE:

> ERP + сервер + 5 лицензий, УТ и КТ Алкоголь

AI автоматически:
1. Вызывает `search_products` для каждой позиции
2. При неоднозначности выбирает наиболее подходящий вариант и помечает флагом
3. Вызывает `build_quote` с найденными кодами и количеством
4. Возвращает готовую таблицу КП с итоговой суммой
