# Разработка и контрибьюция

## Локальная разработка

### Клонировать и установить зависимости

```bash
git clone https://github.com/your-org/1c-products-price.git
cd 1c-products-price
uv pip install -e ".[dev]"
```

### Загрузить прайс для тестов

```bash
mcp-1c-price-update
```

### Запустить тесты

```bash
pytest
```

---

## Архитектура

```
src/mcp_1c_price/
├── settings.py   # конфигурация через переменные окружения
├── db.py         # слой данных: SQLite + FTS5, CRUD-операции
├── updater.py    # загрузка ZIP, парсинг XLS (xlrd), импорт в БД
└── server.py     # MCP-сервер (fastmcp): регистрация и реализация инструментов
```

Зависимости между модулями:

```
server.py → db.py → settings.py
server.py → updater.py → db.py
```

## Добавление нового инструмента

1. Реализуйте логику в `db.py` или `updater.py`
2. Зарегистрируйте инструмент в `server.py` через декоратор `@mcp.tool()`
3. Напишите описание (`description`) на английском — оно читается любой LLM
4. Добавьте тест в `tests/`
5. Обновите `docs/tools.md`

Пример минимального инструмента:

```python
@mcp.tool()
def my_tool(query: str) -> str:
    """
    One-line description in English — what it does and when to call it.
    """
    return db.some_query(query)
```

## Стек

| Компонент | Пакет | Зачем |
|----------|-------|-------|
| MCP-сервер | `fastmcp` | минимум бойлерплейта для MCP stdio |
| HTTP-клиент | `httpx` | скачать ZIP с сервера 1С |
| XLS-парсер | `xlrd` | читать `.xls` (старый формат Excel) |
| Нечёткий поиск | `rapidfuzz` | матчинг при опечатках и сокращениях |
| База данных | `sqlite3` (встроенный) + FTS5 | быстрый полнотекстовый поиск |
| Тесты | `pytest` | |
| Линтер | `ruff` | |

## Источник данных

Прайс-лист скачивается с официального FTP 1С:

```
https://1c.ru/ftp/pub/pricelst/price_1c.zip
```

Файл обновляется ежедневно. Внутри один лист XLS с 9 000+ строк:

| Колонка | Содержимое |
|---------|-----------|
| 0 | Код (артикул) |
| 1 | Наименование |
| 3 | Валюта |
| 4 | Рекомендованная цена |
| 5 | Цена для дилера |
| 6 | Цена для постоянного партнёра |
| 7 | Ставка НДС |
| 8 | Дата начала продаж |
| 9 | Реестр российского ПО |
| 10 | Комментарий |

Строки без числового кода в колонке 0 являются заголовками разделов и пропускаются при импорте.
